# 战斗状态 · 三模块区域尺寸数据（精准参考）

Tailwind 默认：`1 = 0.25rem = 4px`（根字体 16px 时）。`rounded-lg = 8px`。

---

## 一、外层网格（三列容器）

| 项目 | 当前 class | 换算（约） |
|------|------------|------------|
| 列数 | `grid-cols-3` | 三列等分，每列 1fr |
| 列间距 | `gap-3` | **12px** |
| 对齐 | `items-stretch` | 三块等高 |

---

## 二、每个模块卡片（三个共用）

| 项目 | 当前 class | 换算（约） |
|------|------------|------------|
| 圆角 | `rounded-lg` | **8px** |
| 内边距 水平 | `px-1.5` | **6px** |
| 内边距 垂直 | `py-1` | **4px** |
| 边框 | `border` | 1px |
| 最小宽度 | `min-w-0` | 允许在网格内收缩 |

**死亡豁免卡片额外：**
- `w-fit max-w-full md:max-w-[11rem]` → 中屏以上内容最大宽 **176px**（11×16）

## 八、若要“精准修改”时可替换的示例

- **模块内边距更大**：`px-1.5 py-1` → `px-2 py-1.5`（8px / 6px）
- **模块间距更大**：`gap-3` → `gap-4`（16px）
- **标题与内容间距**：`mb-0.5` → `mb-1`（4px）
- **死亡豁免圆点更大**：`w-3.5 h-3.5` → `w-4 h-4`（16px）
- **投掷/重置按钮更大**：`w-7 h-7` → `w-8 h-8`（32px）
- **资源行内间距**：`gap-1` → `gap-1.5`（6px）、`py-0.5` → `py-1`（4px）
- **资源区按钮更大**：`w-6 h-6` → `w-7 h-7`（28px），与投掷按钮一致

按需改对应 class 即可精确控制像素级效果。
