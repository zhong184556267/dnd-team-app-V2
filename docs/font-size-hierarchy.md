# 文字字号层级说明

## 当前使用情况（统一前）

| 字号 | Tailwind | 约像素 | 使用场景 |
|------|----------|--------|----------|
| 最小 | `text-[10px]` | 10px | 战斗手段标题、攻击/伤害/法术等标签、弹窗表单项标签、部分说明 |
| 最小 | `text-[9px]` | 9px | 个别位置（如死亡豁免 d20） |
| 小 | `text-xs` | 12px | 状态效果/死亡豁免/其它职业资源标题与内容、按钮、紧凑区正文 |
| 正文小 | `text-sm` | 14px | 生命值临时说明、主要按钮、弹窗标题与正文、inputStyles 默认 |
| 正文 | `text-base` | 16px | 属性模块等 |
| 大 | `text-xl` | 20px | 生命值数字 |
| 强调 | `text-2xl` | 24px | AC/先攻/被动察觉/速度的标签与数值 |
| 标题 | `text-2xl`/`text-3xl` | 24–30px | 角色名、等级等 |

问题：`text-[10px]` 与 `text-xs`（12px）混用，层级多且不统一。

---

## 统一后的层级（建议）

只保留 **4 档**，避免字号过多：

| 层级 | Tailwind | 约像素 | 用途 |
|------|----------|--------|------|
| **辅助/标签** | `text-xs` | 12px | 区块小标题、表单标签、辅助说明、紧凑区内容、次要按钮 |
| **正文** | `text-sm` | 14px | 正文、主要按钮、弹窗内容、输入框默认 |
| **强调数字/小标题** | `text-base` 或 `text-lg` | 16–18px | 需要略大时使用（可选） |
| **大数字/主标题** | `text-xl` / `text-2xl` / `text-3xl` | 20–30px | 仅用于 HP、AC、先攻等大数字和角色名、等级 |

**具体规则：**
- 所有原 `text-[10px]`、`text-[9px]` 统一为 **`text-xs`**（12px），不再使用 9px/10px。
- 区块小标题（如「状态效果」「战斗手段」）统一 **`text-xs`** + 加粗/大写。
- 弹窗标题保持 **`text-sm`**，弹窗内标签与正文用 **`text-xs`** 或 **`text-sm`** 二选一（当前已按此在 CombatStatus 中统一为 xs）。
- 大数字与主标题保持 **`text-xl` / `text-2xl` / `text-3xl`** 不变。

这样全站最小字号只有 **12px（xs）** 一档，再上一档是 **14px（sm）**，层级清晰、字号数量少。
